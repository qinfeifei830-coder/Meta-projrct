# Meta-project
上海海事大学许峰课题组 Meta 分析研究项目

---

## 📖 项目简介
本仓库为**上海海事大学许峰课题组**的 Meta 分析研究项目开源仓库，旨在系统整合课题组在循证医学、运动科学、公共卫生等领域的 Meta 分析研究成果、分析代码、数据集与技术文档，为相关领域研究者提供可复现、可复用的研究工具与方法学参考。

课题组长期聚焦于**Meta 分析、Meta 回归、网状 Meta 分析、剂量-反应 Meta 分析**等循证研究方法，围绕运动干预、慢性病管理、老年健康等方向开展高水平研究，成果发表于 *Frontiers in Public Health* 等国际知名期刊。

---

## ✨ 最新研究成果
### 1. 运动对阿尔茨海默病老年患者认知功能的影响：Meta 回归与 Meta 分析
- **发表时间**：2025 年 3 月
- **发表期刊**：*Frontiers in Public Health*（JCR Q1，公共卫生领域权威期刊）
- **论文标题**：*Effects of Exercise on Cognitive Function in Elderly Patients with Alzheimer's Disease: A Meta-regression and Meta-analysis*
- **研究亮点**：
  - 系统纳入全球相关随机对照试验（RCT），综合评估运动干预对阿尔茨海默病患者认知功能的改善效果
  - 采用 Meta 回归分析探索干预时长、运动类型、患者基线特征等关键调节因素
  - 为阿尔茨海默病的非药物干预提供高质量循证医学证据，为临床康复方案制定提供参考

## 🛠️ 仓库内容
本仓库将持续更新课题组的 Meta 分析相关资源，包括：
- 已发表 Meta 分析研究的**完整分析代码**（R / Python 实现）
- 标准化的**数据提取模板**与**文献筛选流程**
- Meta 分析方法学教程（如 R 语言 `metafor` / `meta` 包使用、森林图绘制、发表偏倚评估等）
- 课题组最新研究成果、预印本与学术动态
- 可复用的 Meta 分析自动化脚本（批量文献管理、数据合并、结果可视化）
